Documentation

Comment utiliser l'outil avec Hugging Face.

Configuration

  1. Cliquez sur l'icône ⚙️ en haut à droite
  2. Entrez votre token API Hugging Face (créé dans Settings > Access Tokens)
  3. Uploadez une image et choisissez un outil
  4. L'image est envoyée à l'API Hugging Face pour traitement
  5. Téléchargez le résultat

Modèles utilisés

Chaque outil utilise un modèle Hugging Face différent :

Restauration / Amélioration / Défloutage :
caidas/swin2sr-classical-sr-x4-v2
Super-résolution pour améliorer la qualité

Upscaling (4x) :
caidas/swin2sr-classical-sr-x4-v2
Agrandissement 4x par super-résolution

Coloriage :
timbrooks/instruct-pix2pix
Colorisation par édition d'image guidée

Token API

Créez votre token gratuit sur huggingface.co/settings/tokens.
Utilisez un token avec le scope "Inference API".
Le token est stocké uniquement dans votre navigateur (localStorage).
