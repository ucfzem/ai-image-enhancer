Documentation

Comment fonctionne l'outil, ses limites et ses technologies.

Comment ça marche

  1. Uploadez une image (PNG, JPG, WebP)
  2. Choisissez un outil (restauration, coloriage, upscaling, etc.)
  3. L'image est traitée localement ou via l'API Pollinations AI
  4. Le résultat s'affiche avec un comparateur avant/après
  5. Téléchargez le résultat

Outils

OutilBackendType
RestaurationCanvas (median + auto-contrast + sharpen)Local (hors-ligne)
ColoriageCanvas (auto-levels + warm tint + saturation)Local (hors-ligne)
Upscaling (4x)Canvas bicubicLocal (hors-ligne)
DéfloutageCanvas sharpen kernelLocal (hors-ligne)
AméliorationCanvas filtersLocal (hors-ligne)

Limitations

  • Tous les outils fonctionnent hors-ligne (Canvas pur, zéro API)
  • Coloriage : approximation réaliste via auto-levels + teinte chaude
  • Restauration : débruitage + contraste + netteté
  • Taille max : 10MB par image
  • Formats supportés : PNG, JPG, WebP

Technologies

HTML / CSS / JS Canvas API Convolution Kernel 100% Hors-ligne

API Documentation

Intégrez notre outil de traitement d'images dans vos propres applications.

Endpoint

POST /api/process

Paramètres

ParamètreTypeDescription
imageFileFichier image (PNG, JPG, WebP, max 10MB)
toolstringrestore, colorize, upscale, deblur, ou enhance

Exemple (fetch)

const formData = new FormData()
formData.append('image', file)
formData.append('tool', 'restore')

const resp = await fetch('https://ucfzem.github.io/ai-image-enhancer/api/process', {
  method: 'POST',
  body: formData
})

const data = await resp.json()
// { imageUrl: "...", success: true }

Réponse

{
  "success": true,
  "imageUrl": "https://...",
  "tool": "restore",
